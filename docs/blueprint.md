# **App Name**: FiscalFlow

## Core Features:

- User Authentication: Secure user registration and login functionality with role-based access control.
- Fiscal Year Management: Create and manage fiscal year folders to organize financial data.
- Transaction Upload and Integration: Import financial data via bank statements, credit card statements, or direct bank/credit card connection.
- AI-Powered Categorization: Automatically categorize transactions using AI to improve journal entry accuracy using transaction descriptions as a tool.
- Journal Entry Posting: Create and post accurate journal entries to help automate bookkeeping workflows.
- Expense Breakdown Visualization: Visually represent expense breakdowns by category to highlight spending patterns.
- Account Balance Tracking: Monitor and track account balances, presenting a clear financial overview.

## Style Guidelines:

- Primary color: #1E88E5 (primary-600) for main interactive elements, buttons, and primary links.
- Header color: #1565C0 (primary-700) for headers to convey trustworthiness.
- Accent color: #29B6F6 (primary-300) for charts and tags to offer subtle visual cues.
- Background color: #F7FAFC for the overall page to provide a clean slate.
- Surface color: #FFFFFF for cards, panels, and nav bars to ensure content stands out.
- Body and headline font: 'Inter', a sans-serif font known for its clean and neutral appearance, suitable for both body text and headlines in a professional UI. 
- Employ lucide-react icons throughout the app. The icons should be consistent, professional, and intuitive.
- Use a desktop-first responsive design approach that scales well to tablets and phones.
- Use subtle animations and transitions to enhance user experience.
